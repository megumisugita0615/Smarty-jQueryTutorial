# Smarty-jQuery Tutorial

## Smarty

### Env Construction

Clone repository

```
git clone https://github.com/megumisugita0615/Smarty-jQueryTutorial.git
```

move directory

```
cd smarty/
```

Init

```
make install
```

Access `http://localhost/`

```
Hello User, welcome to Smarty!
```
