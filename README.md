- [Docker version 20.10.21, build baeda1f](https://docs.docker.com/engine/release-notes/20.10/)

## Clone repository

```
git clone https://github.com/megumisugita0615/Smarty-jQueryTutorial.git

```
Hello User, welcome to Smarty!
